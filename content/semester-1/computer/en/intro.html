Computer — Introduction (Semester 1)

This subject develops basic computer literacy for D.El.Ed students and introduces how technology can support primary-level teaching.

Key Areas Covered

  • What is a computer: hardware and software basics
  • Input and output devices
  • Operating systems and basic navigation
  • Typing and word processing skills
  • Educational software and digital tools for classrooms
  • Internet safety and responsible use

Classroom Connection

Even simple computer skills like typing, searching, and creating slides help teachers prepare better materials for their students.
