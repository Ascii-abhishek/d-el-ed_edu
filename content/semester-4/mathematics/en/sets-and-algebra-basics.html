1. Set Basics

  • A set is a well-defined collection of objects.
  • Notation: \(A = \{1, 3, 5, 7\}\)
  • \(\varnothing\) means empty set.
  • If every element of \(B\) is in \(A\), then \(B \subseteq A\).
SymbolMeaningExample
\(\cup\)Union: all elements\(\{1,2\} \cup \{2,3\} = \{1,2,3\}\)
\(\cap\)Intersection: common elements\(\{1,2\} \cap \{2,3\} = \{2\}\)
\(\subseteq\)Subset\(\{2,3\} \subseteq \{1,2,3,4\}\)
\(\in\)Belongs to\(3 \in \{1,3,5\}\)

Fast Example

If \(A = \{1,3,5,7,9,11,13\}\) and \(B = \{5,7,11,13\}\), then:

  • \(A \cup B = \{1,3,5,7,9,11,13\}\)
  • \(A \cap B = \{5,7,11,13\}\)
  • \(B \subseteq A\)

2. Indices and Roots

RuleFormulaUse
Product rule\(a^m \cdot a^n = a^{m+n}\)Same base multiply
Quotient rule\(\frac{a^m}{a^n} = a^{m-n}\)Same base divide
Power rule\((a^m)^n = a^{mn}\)Power on power
Fractional power\(a^{m/n} = \sqrt[n]{a^m}\)Roots

Example: \((243)^{3/5} = (\sqrt[5]{243})^3 = 3^3 = 27\), because \(243 = 3^5\).

3. Algebraic Identities

  • \((a+b)^2 = a^2 + 2ab + b^2\)
  • \((a-b)^2 = a^2 - 2ab + b^2\)
  • \((a+b)(a-b) = a^2 - b^2\)

Quick use: \((2+\frac12)(2-\frac12) = 2^2 - (\frac12)^2 = 4 - \frac14 = \frac{15}{4}\).

4. Algebra in One Glance

  • Linear equation: highest power of variable is 1. Example: \(3x = 4.5\)
  • Quadratic equation: highest power is 2. Example: \(x^2 + x - 12 = 0\)
  • Factor method example: \(x^2 + x - 12 = (x+4)(x-3) = 0\)
  • So, \(x = 3\) or \(x = -4\)

Memory Corner

UNION  -> all elements
INTERSECTION -> common elements
SUBSET -> all members already inside bigger set

(a+b)(a-b) = a^2 - b^2
sqrt/root form = fractional power form

Classroom Connection

  • Use colored chalk circles on the floor to show union and intersection physically.
  • Ask children to group classroom objects: red objects, round objects, objects that are both red and round.
  • Teach identities using area models: square of side \((a+b)\).

Exam Lines

  • Union includes all distinct elements of both sets.
  • Intersection includes only common elements.
  • Fractional powers and roots represent the same idea in different notation.
  • Algebraic identities save time and reduce multiplication mistakes.
