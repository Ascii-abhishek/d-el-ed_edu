Periodic Table and Electronegativity

This is a high-scoring chapter. Focus on the history of the periodic table, period/group trends, and the meaning of electronegativity. Exams often ask: differences, definitions, and filling the correct trend.

History of the Periodic Table

  • Döbereiner's Triads (1817): Grouped elements into sets of three with similar properties. Middle element's atomic mass ≈ average of other two. Example: Li, Na, K.
  • Newlands' Law of Octaves (1866): Every 8th element has similar properties to the first (like musical octaves). Failed for elements beyond calcium.
  • Mendeleev's Periodic Table (1869): Arranged 63 known elements by atomic mass. Left gaps for undiscovered elements and predicted their properties. Key exam point: Mendeleev predicted Eka-boron (Sc), Eka-aluminium (Ga), Eka-silicon (Ge).
  • Modern Periodic Table (Moseley, 1913): Elements arranged by increasing atomic number. This solves anomalies in Mendeleev's table.

Period 7:   (Radioactive elements, lanthanides, actinides)
  • 7 Periods (rows): Period 1 has 2 elements; Period 2 and 3 have 8 each; Period 4 and 5 have 18 each.
  • 18 Groups (columns): Group 1 = Alkali metals; Group 2 = Alkaline earth metals; Group 17 = Halogens; Group 18 = Noble gases.
  • s-block: Groups 1 and 2 (very reactive metals + hydrogen, helium)
  • p-block: Groups 13–18 (includes non-metals, noble gases, metalloids)
  • d-block: Groups 3–12 (transition metals — iron, copper, nickel)
  • f-block: Lanthanides and actinides (placed below main table)

Electronegativity: Key Facts

  • Electronegativity = ability of an atom to attract the shared pair of electrons in a covalent bond toward itself.
  • Measured by Pauling scale.
  • Fluorine (F) has the highest electronegativity (4.0) — top right of the periodic table.
  • Francium (Fr) has the lowest electronegativity — bottom left.
  • If the difference in electronegativity between two atoms is large → ionic bond forms.
  • If the difference is small → covalent bond forms.
  • Use this to predict bond type in exam questions.

Common Exam Lines (Direct Use)

  • Mendeleev arranged elements in order of increasing atomic mass; Modern table uses atomic number.
  • Elements in the same group have the same number of valence electrons.
  • As we go left to right in a period, metallic character decreases and non-metallic character increases.
  • Noble gases (Group 18) have complete outer shells and are chemically inert.
  • The atomic number of an element = number of protons = number of electrons (in neutral atom).

Classroom Connection

  • Use a large printed periodic table poster. Have students colour metals (gold), non-metals (blue), and metalloids (green).
  • Ask: "Why does sodium react with water but argon does not?" — this leads to Group 1 vs Group 18 valence electrons.
  • Make element cards with name, symbol, atomic number — sort by property, not by rote.

One-minute Revision

  • Period = horizontal row = same number of shells.
  • Group = vertical column = same valence electrons = similar properties.
  • Atomic radius: increases down a group, decreases across a period.
  • Electronegativity: highest at top-right (Fluorine), lowest at bottom-left (Francium/Caesium).
  • Mendeleev → atomic mass; Modern table → atomic number.
