# Periodic Table and Electronegativity

This chapter introduces the periodic table and the basic idea of electronegativity in simple classroom language.

## Key Points

- Elements are arranged in the periodic table according to their properties.
- Groups contain elements with similar chemical behavior.
- Periods show gradual changes in properties from left to right.
- Electronegativity means the tendency of an atom to attract shared electrons.

## Classroom Connection

Use element cards to help students sort elements into broad groups and observe patterns.
